Project Description

This is a personal project in development focused on learning about web development through ruby on rails. To begin understanding the language I am incorporating social media aspects with the possibility for other features. The base includes a home page with login and sign up features. The website contains a database through SQL that will avoid duplication of emails. The project then has a posts page that allows unsigned in users to view posts without the ability to modify or add posts themselves. This also means that when logged in posts are tied to the user and only the creator may modify those posts.

As the project currently stands there are features that are not perfectly fluid and the page is not fully modular to allow for mobile use. As this is a creative project the addition of new features are not bound by site specificity. Sporcle and other trivia websites are possible inspirations for different paths this project might take.

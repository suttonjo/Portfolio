Image Super Resolution

December 2023

This project replicates super resolution neural networks through MATLab in order to find layers that could be modified in order to avoid exceding memory capacity while train convolutional neural networks. Despite being slower, LESRCNN showed superior results, although still below the original papers' outcomes. Challenges included Matlab's Deep Learning Toolbox intricacies and customizing layers for optimal performance. Ultimately, their LESRCNN outperformed others in PSNR, showcasing significant advancements in image super-resolution techniques.

Viking Ship VR Assessment

May 2024

This project had the purpose to give developing GCIEL Viking Ship VR learning Experience team ways to make sense of player data. The project is meant to create new learning experiences for the 21st century. The expansion of the project could lead to recreating the Shakespeare Globe Theatre in order to teach how it was used through a VR experience. Currently the project scope is to create a learning experience in how Viking ships were made and how riding a ship might've been like. In order to find good features or find where features might need to be added in order to make the learning experience as user friendly as possible. There is data taken from each experience and the goal of our app was to create ways to understand multiple user experience easily and to think of new advancements for the experience.
